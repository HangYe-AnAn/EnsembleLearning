# Ensemble Learning R Package

## Quick start

### Step 1: Installing dependency

- installing "devtools" by using the `install.packages("devtools")` command
- use `install.packages("e1071")` to install the "e1071" package
    
### Step 2: Install the Package in R:
- Open R or RStudio.
Use the command `devtools::install_github("HangYe-AnAn/EnsembleLearning")` to install the package

### Step 3: loading the library:
- Load the installed package using the `library()` function.
- Verify that the package loads without errors.
`library(simpleEnsembleGroup7)` 

### Step 4: Open the tutorial
- using the command `?simpleEnsembleGroup7` to open tutorial page
